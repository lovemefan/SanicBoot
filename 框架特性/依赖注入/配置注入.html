配置注入@Value#

class backend.core.component.value.Value(key, default=None)[source]#

Used for injecting values into fields from config.yaml

Parameters:

  • key – the key of config file, such as server.http.port

  • default – default value, default None

@Value的作用是将配置文件的值注入到代码文件当中 由于python装饰器只支持装饰类和方法，没有办法像java一样装饰变量。 @value参考了python内置的@property装饰器，通过装饰类方法实现类成员变量的注入。

 1class configTest:
 2    @Value('datasource.mysql.port')
 3    def port(self):
 4        pass
 5
 6    def connect():
 7        port = self.port
 8
 9config = configTest()
10print(config.port)

还有另外两种获取配置文件 @Value的使用。
