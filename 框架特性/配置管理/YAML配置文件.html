获取配置项#

获取本地配置有三种方法：

1. 继承BaseConfig类#

class backend.config.BaseConfig.BaseConfig[source]#

implements this class to use the get method to get config from config file Example : class A(BaseConfig) use a = A() ,you can use a.get(key:str) to get config from config file

static get(key, default=None)[source]#

get value from config by the key file dynamically

Parameters:

  • key (str) – the key of config file, example host.ip.

  • default (str) – if the key is not exist,return default as value.

Returns:

the value of config file key example : if the key is host.ip maybe it will return localhost

Return type:

any

使用方法#

1from backend.config.BaseConfig import BaseConfig
2
3class Mysql(BaseConfig):
4    def __init__():
5        self.host = self.get("datasource.mysql.host", 'localhost')
6        self.port = self.get("datasource.mysql.port", 80)

2. 使用工具类获取#

class backend.config.Config.Config(config_file_path=None)[source]#

upgrade the config automatically while the config.ini file changed Example Config.get_instance().get(key, default)

get(key, default=None, config=None)[source]#

get value by the key from config

Parameters:

  • key (str) – format [section].[key] example: app.name

  • config (dict) – config

  • default – if the key not exist ,return default value

Returns:

value of key, if the key not exist ,return default value

Return type:

str

static get_instance()[source]#

get a instance at once simultaneously

load_config()[source]#

load the config file

使用方法#

1from backend.config.Config import Config
2host = Config.get_instance().get("datasource.mysql.host", "localhost")
