sanic-backend
=============

Sanic-backend是基于sanic封装的python web快速开发框架。

.. attention::
    最低依赖要求：

    - Python: 3.8+
    - Sanic: 21.9+


- 简单轻便
    - 开箱即用，像spring boot一样快速构建属于您的API 应用程序
- 易于拓展
    - 具备sanic优秀的拓展性，随时可以为各种大小的 Web 应用程序提供支持。
